# Introduction
A level gauge, designed to read level in millimetres from an RCWL-9610A ultrasonic distance sensor in UART mode.

It is best to open this project in Visual Studio Code using the `Open Workspace from File` option and selecting [`level_gauge.code-workspace`](level_gauge.code-workspace): that way it will open not only the current project but also the components that the project may depend on.
